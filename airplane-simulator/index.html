AirportSim

A terminal-based airport simulator written in C++17 from scratch. Manages real-time plane arrivals, departures, and fuel emergencies using manually implemented data structures — no STL queues. Features a live ANSI terminal display with colour-coded status, fuel bars, and a scrolling event log.

Priority Queue
Sorted linked list for arrivals. Emergency planes (<20% fuel) jump to the front. Insertion O(n), dequeue O(1).
→
FIFO Queue
Standard linked list for departures. First in, first out. Enqueue and dequeue both O(1). No STL container used.
🛬
Fuel Emergencies
Arriving planes burn 1% fuel per minute while waiting. Below 20% triggers EMERGENCY priority. At 0% — crash.
🛫
Runway State Machine
Each runway cycles: AVAILABLE → LANDING/TAKEOFF → AVAILABLE. Arrivals always take priority over departures.
